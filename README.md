<h2>Rentify – Property Booking Platform</h2>

📌 Overview:
A full-stack web application that allows users to list, browse, and book rental properties—similar to Airbnb.

❗ Problem:
Finding and managing short-term rental properties can be inefficient due to:
* Lack of centralized platforms
* Poor user experience
* Limited booking management systems

💡 Solution
StayEase provides a seamless platform where:
* Hosts can list properties
* Users can explore listings
* Bookings can be managed efficiently

🛠️ Tech Stack:
* Frontend: React (Vite)
* Backend: Node.js, Express
* Database: MongoDB
* Authentication: JWT
* Styling: Tailwind CSS

✨ Features:
* 🔐 User Authentication (Login/Register)
* 🏠 Property Listings (Add / Edit / Delete)
* 🔍 Browse & Search Listings
* 📅 Booking System
* 📊 User Dashboard
